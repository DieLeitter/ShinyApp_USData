USA Data Shiny App

This shiny app generates scatter plots between two chosen variables and fit a linear regression. The varaibles are selected from state.x77 dataset (more info about data set here https://stat.ethz.ch/R-manual/R-devel/library/datasets/html/state.html). One can choose between 8 features: Population, Income, Illiteracy, Life Exp., Murder, HS Grad, Frost, and Area.

To get started:

  • Select desired x and y from the input boxes.
  • Brush the plot to select points for which you would like to fit regression line. The regression line will be added as a blue line in the plot. Note: at least two points need to be selected to fit a regression line.
  • Zoom in can be done by double clicking on previously selected (brushed) area.
